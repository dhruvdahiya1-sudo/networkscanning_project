# 🔍 Network Scanner using Python

Welcome to my **first GitHub project**! This is a simple yet powerful **Network Scanning tool** built using **Python, sockets, and Scapy**. It helps identify active devices, open ports, services, and even attempts OS fingerprinting on a given network or domain.

---

## 🚀 About the Project

This project is a beginner-friendly **network scanning tool** designed to deepen understanding of how devices communicate over a network. It performs:

- IP address discovery using **Scapy**
- Port scanning via **TCP sockets**
- Basic **service enumeration** (like detecting HTTP, FTP, etc.)
- **OS fingerprinting** using TCP/IP stack behavior

A perfect entry-level tool for anyone learning networking or ethical hacking!

---

## 🧰 Features

- ✅ Hostname to IP resolution
- ✅ IP discovery within subnet using Scapy
- ✅ Custom port range scanning
- ✅ Open/closed port detection
- ✅ Basic service detection (via common port lookup)
- ✅ OS fingerprinting (experimental)
- ✅ Error handling for invalid input
- ✅ Clean, beginner-friendly CLI output

---

## 🧪 Tools & Libraries Used

- socket – for low-level network communication
- scapy – for packet crafting and subnet IP discovery
- ipaddress – for subnet parsing and manipulation
- threadigng - to use multithreading concept for concurrent execution of scaninng
